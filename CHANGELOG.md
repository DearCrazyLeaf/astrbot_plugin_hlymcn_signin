# 更新日志

本文档记录 `astrbot_plugin_hlymcn_signin` 的版本变更。

## v0.8.0

### 新增
- 新增查服背景模式配置 `server_query_bg_mode`：
- `header`：使用插件自定义头图
- `map_cover`：优先使用后端地图封面
- 新增后端地图封面字段兼容解析：
- `mapImage`
- `map_image`
- `basic_info.map_image`

### 优化
- 查服 `card` 与 `image_text` 的背景选择逻辑统一。
- `map_cover` 模式下，地图封面不可用时自动回退到自定义头图。
- `map_cover` 模式下减少不必要的头图预载/静默刷新操作。
- README 重构：补充依赖矩阵、两种部署路径、接口契约与配置盲区说明。

### 变更
- 插件版本更新为 `v0.8.0`。
