# 更新日志

本文档记录 `astrbot_plugin_hlymcn_signin` 的版本变更。

## v0.8.1

### 新增
- 增加地图封面本地缓存目录 `map_cover_cache`（位于 AstrBot 规范数据目录）

### 优化
- `map_cover` 模式优先读取本地地图封面缓存
- 缓存未命中时拉取封面并按地图名保存，加快后续同地图出图速度
- 增加地图封面缓存清理策略（默认保留最近 200 张）

### 变更
- 插件版本更新为 `v0.8.1`

## v0.8.0

### 新增
- 新增查服背景模式配置 `server_query_bg_mode`
- `header`：使用插件自定义头图
- `map_cover`：优先使用后端地图封面
- 新增后端地图封面字段兼容解析：
- `mapImage`
- `map_image`
- `basic_info.map_image`

### 优化
- 查服 `card` 与 `image_text` 背景逻辑统一
- `map_cover` 模式下地图封面不可用时自动回退到自定义头图
- `map_cover` 模式下减少不必要的头图预载与静默刷新
- README 补充依赖矩阵、部署路径、接口契约与配置说明

### 变更
- 插件版本更新为 `v0.8.0`
